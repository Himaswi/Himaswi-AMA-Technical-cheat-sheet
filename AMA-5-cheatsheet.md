### 1.How do you start a project in Django?
Create a project using: django-admin startproject project_name

### 2.What is models.Model?
models.Model is the base class used to define database tables as Python classes in Django.

### 3.Why do we use venv?
venv is used to create an isolated Python environment to manage project-specific dependencies.

### 4. What are Django apps?
Django apps are modular components that contain specific functionality like models, views, and templates.

### 5. What is an HTML template in Django?
An HTML template is a dynamic HTML file that uses Django Template Language (DTL) to render data.

### 6. What are serializers in Django?
Serializers convert complex data (models/querysets) into JSON and vice versa, mainly used in Django REST Framework.

### 7.What is middleware in Django?
Middleware is a layer that processes requests and responses globally before views and after responses.

### 10. What is a function in Python?
A function is a reusable block of code defined using the def keyword that performs a specific task.
